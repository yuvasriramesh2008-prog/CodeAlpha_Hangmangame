# CodeAlpha Python Programming Internship — Task 1: Hangman Game

A simple text-based Hangman game built in Python.

## Features
- Random word selection from a predefined list of 5 words
- Limits players to 6 incorrect guesses
- Basic console input/output, no external libraries required

## Concepts Used
`random`, `while` loop, `if-else`, strings, lists, functions

## How to Run
```
python hangman.py
```

## Sample Gameplay
```
Welcome to Hangman!
You have 6 incorrect guesses allowed.

Word: _ _ _ _ _ _
Wrong guesses left: 6
Guess a letter: p
Good guess! 'p' is in the word.
...
Word: p y t h o n
Congratulations! You guessed the word: python
```
